Cài đặt môi trường bằng các lệnh sau:

npm init -y
npm install cypress --save-dev
npm install -D cypress-plugin-tab
    - Thêm vào đầu file cypress lệnh: require("cypress-plugin-tab");
npm i --save-dev cypress-mochawesome-reporter
npm install xlsx


Bổ sung:
    File-Excel (Là thư mục có chứa file excel quan trọng để chạy tool)
